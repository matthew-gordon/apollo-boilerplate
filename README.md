# apollo-boilerplate

setup and run server first
## Server

`$ export DATABASE_URL=postgres://localhost:5432/<db_address>`

`$ export TOKEN_SECRET=changeme`

`$ cd server`

`$ yarn install`

`$ yarn migrate`

`$ yarn seed`

`$ yarn dev`

then start the client
## Client

`$ cd client`

`$ yarn install`

`$ yarn start`
